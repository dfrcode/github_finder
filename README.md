# Github Finder

### 1. Using
```
- react;
- react routing;
- react hooks;
- axios.
```

### 2. Init
```
~$ cd github_finder/
~$ npm run install
~$ npm run start
```
### 3. Adding
```
~$ touch .env.local
~$ echo "REACT_APP_GITHUB_FINDER_CLIENT_ID='client_id'" >> .env.local
~$ echo "REACT_APP_GITHUB_FINDER_CLIENT_SECRET_ID='client_secret'" >> .env.local
```
### 4. Deploy
```
~$ npm run build
```
